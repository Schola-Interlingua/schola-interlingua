## 🚀 Nueva Versión React

**Schola Interlingua** ha sido completamente modernizado y convertido a **React 18** con una interfaz de usuario completamente nueva y moderna. Esta nueva versión incluye:

### ✨ Características Principales

- **🎨 Diseño Moderno**: Interfaz completamente rediseñada con Tailwind CSS
- **📱 Responsive**: Optimizado para todos los dispositivos
- **🎭 Animaciones**: Transiciones suaves con Framer Motion
- **🎯 Componentes React**: Arquitectura modular y reutilizable
- **🌈 Sistema de Colores**: Paleta moderna y accesible
- **📊 Dashboard Interactivo**: Seguimiento de progreso en tiempo real
- **🔍 Búsqueda Avanzada**: Filtros y búsqueda inteligente de lecciones
- **📈 Métricas Visuales**: Gráficos y estadísticas atractivas

### 🛠️ Tecnologías Utilizadas

- **Frontend**: React 18 + Hooks
- **Styling**: Tailwind CSS + CSS Modules
- **Animaciones**: Framer Motion
- **Iconos**: Lucide React
- **Build Tool**: Create React App
- **Fonts**: Inter, Poppins, JetBrains Mono

## 🚀 Instalación y Configuración

### Prerrequisitos

- Node.js 16+ 
- npm o yarn

### 1. Clonar el Repositorio

```bash
git clone https://github.com/schola-interlingua/schola-interlingua.git
cd schola-interlingua
```

### 2. Instalar Dependencias

```bash
npm install
# o
yarn install
```

### 3. Ejecutar en Desarrollo

```bash
npm start
# o
yarn start
```

La aplicación se abrirá en [http://localhost:3000](http://localhost:3000)

### 4. Construir para Producción

```bash
npm run build
# o
yarn build
```
