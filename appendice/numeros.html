Regulas de formation

  • Le cifras de zero a novem ha formas specific: zero, un, duo, tres, quatro, cinque, sex, septe, octo, nove.
  • Le dezenas se forma con le cifra + "anta", excepte dece [10] e vinti [20]: tresanta, quaranta, cinquanta, sexanta, septanta, octanta, novanta.
  • Numeros composte se forma con un trait-d'union: dece-duo, septanta-nove.
  • Le centos se forma con le cifra + "centos": cento, duo centos, tres centos, etc.
  • Le milles se forma similar: mille, duo milles, tres milles, etc.
  • Le scala longe: million, milliardo, billion, billiardo, trillion, trilliardo.
