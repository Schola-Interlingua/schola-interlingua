Regulas de formation

  • Le cifras de zero a novem ha formas specific: zero, un, duo, tres, quatro, cinque, sex, septe, octo, novem (o nove; le -m final del latino classic es frequentemente usate pro evitar le confusion con le adjectivo nove que es le contrario de ancian).
  • Le prime decenas: dece [10], vinti [20], trenta [30], quaranta [40].
  • Le decenas a partir de 50 se forma con le cifra + "anta": cinquanta, sexanta, septanta, octanta, novanta.
  • Le numeros composite se forma con un tracto de union: dece-duo, septanta-novem.
  • Le centos se forma con le cifra + "centos": cento, duo centos, tres centos, etc.
  • Le milles se forma de maniera similar: mille, duo milles, tres milles, etc.
  • Es usate le scala longe: million, milliardo, billion, billiardo, trillion, trilliardo.
